Este programa implementa um TAD para manipulação de uma coleção generica.
Propriedade de: ROCHEL ANDREY DOS SANTOS COSTA
MATRICULA: 2023090100

1 Ambiente ao qual o programa foi desenvolvido:
- SO: Windows 11
- Compilador: MinGW, GCC
- Linguagem: C
- IDE: CLion e Prompt de Comando

2 Arquivos
- colecao.h: define os protótipos e os cabeçalhos que vão ser utilizados no arquivo de implementação
- colecao.c: Implementação das funções
- ColMain.c: Arquivo principal, função main com todo o programa utilizando as funções descritas no enunciado
- TadColecao.exe: Arquivo executável gerado após a compilação do código.

 Formas de executar:

 -A. Executando diretamente pelo CMD:
1. Abra o Prompt de Comando, digite cd (CaminhoDoDiretório) e acesse o local onde está a pasta do programa.
2. Escreva o nome do executável e deve executar normalmente
3. Dê duplo clique no executável, com a pasta aberta

OBS: Caracteres como "ã" ou "~~" "´´" não são mostrados corretamente
