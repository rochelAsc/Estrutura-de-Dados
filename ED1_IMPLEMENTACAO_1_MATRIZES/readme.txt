Este programa implementa um TAD para manipulação de matrizes.
Propriedade de: ROCHEL ANDREY DOS SANTOS COSTA
MATRICULA: 2023090100

1 Ambiente ao qual o programa foi desenvolvido:
- SO: Windows 11
- Compilador: MinGW, GCC
- Linguagem: C
- IDE: SublimeText(custom build, não compilado como .exe), CLion(build padrão, não compilado como .exe), Command Prompt(linkagem dos dois arquivos '.c' e executado)

2 Arquivos
- matriz.h: Define a estrutura e prototipo das funções para manipulação de matrizes, como criação, destruição e acesso a elementos.
- matriz.c: Implementação das funções de manipulação de matrizes.
- apl.c: Arquivo principal, função main com todo o programa utilizando as funções descritas no enunciado
- apl.exe: Arquivo executável gerado após a compilação do código.

 Formas de executar:

 -A. Executando diretamente pelo CMD:
1. Abra o Prompt de Comando, digite cd (CaminhoDoDiretório) e acesse o local onde está a pasta do programa.
2. Para compilar num arquivo só, escreva gcc -o apl.exe apl.c matrix.c -I.
3. Escreva apl.exe para executar o programa (enquanto voce ainda estiver no diretório)

 -B. Executando diretamente pela pasta, dê duplo clique no apl.exe e veja a magica acontecer (deve acontecer)

OBS: Caracteres como "ã" ou "~~" "´´" não são mostrados corretamente, isso é normal
